# Proyecto-de-Pruebas-para-Urban-Scooter---Proyecto-Final
Probar la aplicación Urban Scooter, un servicio que permite alquilar un scooter eléctrico durante varios días.  Diseñar y ejecutar casos de prueba funcionales para una aplicación de transporte urbano (Scooter eléctricos), validando flujos de reserva y seguimiento en tiempo real.
